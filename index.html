Polus Quiz-building Website

Polus launched a website that was able to provide students a study option that prizes customizability and simplicity in design. As Polus develops more quizzes, the modular design of the website’s quiz function makes implementing new quizzes simple.

As the focus of the website was the quiz function, I first had to design the mechanism in which the quiz would function. Since it was to be designed entirely in JavaScript, I began researching strategies I could implement in the design.

Alongside mapping out the functionality of the quizzes, I was tasked with designing the general branding of the website. After examining Polus’s missions and exploring the branding of similar sites on the market, I developed a visual design that satisfied the academic identity of Polus while maintaining a calm environment for the quiz-takers.

It was important that the site could implement more quizzes at late dates, so I designed the quiz function to be able to load pre-made quizzes in a JSON format. This way, Polus could design new quizzes, and just direct the quiz function to that new document. Additionally, this opened up the ability to minimize the number of pages the website would have to host – including the need for a separate page for user generated quizzes.

Games Done Quick convention collateral

GDQ’s summer program was in need of branded conference collateral. Convention goers were supplied with convention materials that exhibited the spirit of the convention, clearly dictated the event’s information, and identified official functions.

Beginning the project, I drafted out ideas for the layouts of the three different collaterals, with the greatest focus being placed on the conference’s program. I then collected all the assets I would need from official sources – the branding guide, the conference’s information and event schedule, photos of the speakers, and images from past conferences.

Utilizing GDQ’s branding guide, I developed the program in Adobe InDesign. With the conference’s program design fully realized, I developed the remaining collateral – the conference badges and table tents – to be fully compliant with the program’s design, and ensure a consistent branding across each element.

Amazon Kindle App UI/UX redesign

Amazon’s Kindle app needed an update to streamline the user experience. With the new UI design, the kindle app became more accessible to the average user, and by utilizing a more tactile-inspired approach that invoked the imagery of flipping through books in a library,

I began by analyzing the Kindle app and investigating where improvements would likely be made. After exploring similar apps with library functions for inspiration, I designed a series of static drafts to serve as the storyboard for my prototype.

Based on my storyboard and in line with Amazon’s brand guidelines, I created assets in Adobe Illustrator that I would then import into Adobe After Effects, the program I would create the prototype in.

Following my storyboard, I created the motion prototype in After Effects, making minor adjustments to the Illustrator assets as the project completed.

About
